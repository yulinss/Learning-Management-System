## Run Locally

Clone the project

```bash
  git clone https://github.com/abdulaziz-m5u/course-LMS.git your-project
```

Go to the project directory

```bash
  cd your-project
```

-   Copy .env.example file to .env and edit database credentials there

```bash
    composer install
```

```bash
    php artisan key:generate
```

```bash
    php artisan db:seed
```
